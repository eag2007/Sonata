### Глава 8. Потоки
Язык Sonata поддерживает "цепочки действий", называемыми потоками. Они служат чтобы ускорить написание кода,
упростить синтаксис, повысить читаемость. Все это осуществляется с помощью набора команд и оператора 
<code>-></code>.<br><br>
Покажем на примере<br>
Задача: найти все элементы списка большие 4, кратные 5. Домножить их на 6 и прибавить 7. Вывести на эран<br>
<code>data = [12, 1, 34, 2, 5, 10, 11, 2, 4, 0]<br>
print(data -> filters(_ > 4) -> filters(_ % 5 == 0) -> operation(_ * 6 + 7))<br>
</code><br>
Выведет: <code>[37, 67]</code><br><br>

Основные команды при работе с потоками:
<br><code>filters()</code> - принимает условие и отбирает элемент согласно этому условию
<br><code>operation()</code> - принимает правило операции и применяет его к каждому элементу
<br><code>level: ?</code> - указывает какой уровень вложенности брать
<br><code>groupBy()</code> - группирует элементы (<code>['a', 'b', 'a'] -> ['a': ['a', 'a'], 'b': ['b']])</code>
