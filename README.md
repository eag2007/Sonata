### Глава 1. Язык Sonata
Язык Sonata является динамическим потоковым языком с универсальной типизацией. 
Язык создавался с целью ускорения написания кода, логики программы, одешевлением в разработке.
Является полностью Open Source прооектом. Дата задумки языка: ноябрь 2025
